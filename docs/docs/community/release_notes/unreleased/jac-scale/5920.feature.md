- **Feat: MongoDB filter pushdown for `[?:Type, field op const]` edge traversals**: `MongoBackend.query_by_type` now accepts an optional `scoped_ids` list. When jaclang's runtime detects a `FilterSpec`-wrapped filter on a single-hop traversal, it calls `query_by_type` with the predicate dict and neighbour IDs as a `$in` constraint, running the query entirely server-side. Requires jaclang 0.14.2+.
